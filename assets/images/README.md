# Image Assets Folder

Place your website images in this folder:

## Required Images:

1. **background.jpg** (1920x1080 recommended)
   - Hero section background image
   - Should be visually appealing and professional
   - Can be a gradient, workspace, or abstract background

2. **profile.jpg** (500x600 recommended)
   - Your professional photo for the About section
   - Should have good lighting and be well-composed
   - Square or portrait orientation works best

3. **work1.jpg** (600x400 recommended)
   - Screenshot of your first project
   - Should showcase the project's interface or key features

4. **work2.jpg** (600x400 recommended)
   - Screenshot of your second project

5. **work3.jpg** (600x400 recommended)
   - Screenshot of your third project

## Image Tips:

- **Optimize your images** before uploading using [TinyPNG](https://tinypng.com)
- Keep file sizes under 500KB each for fast loading
- Use high-quality images that represent your work professionally
- Maintain consistent aspect ratios for project screenshots
- Consider using free stock photos from [Unsplash](https://unsplash.com) or [Pexels](https://pexels.com) as placeholders

## Free Image Resources:

- **Backgrounds:** Unsplash, Pexels, Pixabay
- **Tech/Workspace:** Use search terms like "developer workspace", "coding", "tech background"
- **Profile:** Use a professional headshot or professional photo

## File Naming:

Make sure your image file names match exactly:
- `background.jpg` (lowercase)
- `profile.jpg` (lowercase)
- `work1.jpg` (lowercase)
- `work2.jpg` (lowercase)
- `work3.jpg` (lowercase)

The website is case-sensitive, so use lowercase file names.
